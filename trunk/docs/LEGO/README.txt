                    LEGO Mindstorms NXT Documentation

This directory contains the LEGO Mindstorms NXT documentation for the
Bluetooth communication protocol and hardware interfaces and schematics.  This
documentation is, however very large and copywrited, so it is not checked in.

The documentation can be retrieved here:

http://mindstorms.lego.com/Overview/nxtreme.aspx
    * Bluetooth Developer Kit (BDK)     - Bluetooth communication protocol
    * Hardware Developer Kit (HDK)      - Hardware interfaces and schematics
